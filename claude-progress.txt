## 완료
- 베이스라인(Qwen2.5-VL-3B) → H100 노트북(Qwen2.5-VL-7B) 하이퍼파라미터 튜닝
  - 모델: 3B → 7B (H100 80GB VRAM 활용)
  - 이미지: 384px → 672px (재활용 이미지 세부 특징 인식 향상)
  - LoRA: r=8/alpha=16 → r=64/alpha=128
  - LR: 1e-4 → 1e-5, Epoch: 1 → 3 + cosine scheduler
  - FULL_DATA_MODE: train+dev 전체 데이터 학습
  - 앙상블용 Model B 추가 (seed=123, lr=5e-5)
- STRATEGY.md 기반 발표자료 + 표지 리디자인 완료
- 72B Zero-Shot 시도 → **실패 (dev acc = 0.4618)** → 포기
- **7B Multi-Diversity Ensemble 노트북 완성** (`aggressive_final.ipynb`)
  - 72B 의존성 완전 제거, 7B 중심 전략으로 전환
  - 3가지 프롬프트(Original/Detailed/Minimal) × dev 비교 → 최적 자동 선택
  - 12-permutation TTA (기존 4에서 3배 확장)
  - Model A + Model B 앙상블 (seed/LR 다양성)
  - Model A best + Model A alt prompt 앙상블 (프롬프트 다양성)
  - 3-way 앙상블 (A_best + A_alt + B)
  - 3가지 앙상블 방법: softmax weighted, confidence-weighted, 3-way
  - Dev 기반 가중치 최적화 (9-point grid × 3-way grid)
  - 10+ submission 변형 자동 생성
  - 코드 리뷰 완료: 메모리 관리, JSON 호환성, 프롬프트 일관성 모두 검증
